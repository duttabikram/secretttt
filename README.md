# secretttt
A minimal HTML project containing a single `index.html` file.

## Installation
1. Clone the repository  
   ```bash
   git clone https://github.com/duttabikram/secretttt.git
   ```
2. Navigate to the project folder  
   ```bash
   cd secretttt
   ```

## Usage
Open the `index.html` file in any web browser:

```bash
# On macOS / Linux
open index.html

# On Windows
start index.html
```
